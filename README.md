# Demo code and slides for "React at 60 FPS"

Thanks for your interest in animating in React!

To run the demo: clone the repo, run `npm install`, then `npm run dev`. Vite should open the app on port 5173.

Slides are inside the repo under `slides.` You can also launch them from the Vite app by clicking the "open slides" button.
